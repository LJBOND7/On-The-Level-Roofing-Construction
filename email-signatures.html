How to use these

  1. Pick a signature block below.
  2. Click and drag to select the whole block, then Copy (Cmd C).
  3. Mail.app: Mail menu, Settings, Signatures. Pick the account, click +, then Paste (Cmd V). Uncheck "Always match my default message font."
  4. Gmail web: Settings, See all settings, General, Signature, Create new, then Paste.
  5. Swap the phone number once your Google Voice line is set up.
